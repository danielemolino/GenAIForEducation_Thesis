# Setup iniziale (prima volta)

## 0) Prerequisiti2 e install dipendenze
```bash
# Node/Yarn (monorepo OHIF + lerna)
node -v
yarn -v
cd /mnt/d/GenAIforEdu/Viewer
yarn config set workspaces-experimental true
yarn install

# Python backend
cd /mnt/d/GenAIforEdu
python -m venv .venv
source .venv/bin/activate   # su PowerShell: .venv\Scripts\Activate.ps1
pip install --upgrade pip setuptools wheel
pip install numpy pydicom fastapi nibabel pillow tifffile uvicorn
```

## 0b) Setup Python su GN03 (Conda/Miniforge, no Python 3.6 system)
```bash
module load amd/gcc-8.5.0/miniforge3

conda create --name genedu_cuda python=3.10.4 -y
source activate genedu_cuda

pip install --upgrade pip setuptools wheel
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu126
pip install fastapi uvicorn numpy pydicom nibabel pillow tifffile

# check
python --version
python -c "import torch; print(torch.__version__, torch.cuda.is_available())"
```

# Avvio rapido - GENEDU (VM web)

Tre terminali separati su `genedu`:

## 1) Orthanc (Docker)
```bash
cd /mnt/d/GenAIforEdu/Viewer
yarn orthanc:up
```

## 2) Viewer (web app)
```bash
cd /mnt/d/GenAIforEdu/Viewer
yarn dev:orthanc
```

## 3) Backend orchestratore (chiama servizi remoti)
```bash
cd /mnt/d/GenAIforEdu/BackendModelli
pip install numpy pydicom fastapi nibabel pillow tifffile uvicorn
export BACKEND_MODE="full"
export GENERATION_ENGINE="real"
export REMOTE_INFERENCE_URL="http://slurmsrv:4450"
uvicorn main:app --host 0.0.0.0 --port 8000 --access-log
```

# Servizi modello su GN03 + gateway unico su 8001

Architettura:
- `genedu` chiama `http://slurmsrv:4450`
- `slurmsrv:4450 -> gn03:8001` (gateway)
- gateway inoltra localmente a:
  - CT worker `127.0.0.1:8002`
  - XGeM legacy gia attivo `127.0.0.1:8000`

## 1) XGeM legacy (gn03:8000)
```bash
# Gia attivo per altra web page (non avviare una seconda istanza sulla stessa porta)
curl http://127.0.0.1:8000/healthz
```

## 2) CT worker (gn03:8002)
```bash
cd /path/to/GenAIForEducation_Thesis/BackendModelli
module load amd/gcc-8.5.0/miniforge3
source activate genedu_cuda
uvicorn remote_services.workers.ct_worker:app --host 0.0.0.0 --port 8002 --access-log
```

## 3) Gateway unico (gn03:8001)
```bash
cd /path/to/GenAIForEducation_Thesis/BackendModelli
module load amd/gcc-8.5.0/miniforge3
source activate genedu_cuda
export CT_WORKER_URL="http://127.0.0.1:8002"
export XRAY_WORKER_URL="http://127.0.0.1:8000"
export XRAY_HEALTH_PATH="/healthz"
export XRAY_INFER_PATH="/generate"
export XRAY_LEGACY_TASK="T->F"
uvicorn remote_services.gateway.main:app --host 0.0.0.0 --port 8001 --access-log
```
