# Setup iniziale (prima volta)

## 0) Prerequisiti2 e install dipendenze
```bash
# Node/Yarn (monorepo OHIF + lerna)
node -v
yarn -v
cd /mnt/d/GenAIforEdu/Viewer
yarn config set workspaces-experimental true
yarn install

# Python backend
cd /mnt/d/GenAIforEdu
python -m venv .venv
source .venv/bin/activate   # su PowerShell: .venv\Scripts\Activate.ps1
pip install --upgrade pip setuptools wheel
pip install numpy pydicom fastapi nibabel pillow tifffile uvicorn
```

# Avvio rapido

Tre terminali separati:

## 1) Viewer (web app)
```bash
cd /mnt/d/GenAIforEdu/Viewer
yarn dev:orthanc
```

## 2) Docker (Orthanc)
```bash
cd /mnt/d/GenAIforEdu/Viewer
yarn orthanc:up
```

## 3) Backend principale unico (router CT/XRay)
```bash
cd /mnt/d/GenAIforEdu/BackendModelli
pip install numpy pydicom fastapi nibabel pillow tifffile uvicorn
$env:BACKEND_MODE="full"
$env:GENERATION_ENGINE="real"
uvicorn main:app --host 0.0.0.0 --port 8000 --access-log
```
